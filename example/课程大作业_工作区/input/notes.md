# 课堂笔记摘录

## 第 1 周：生成式 AI 基础
- 大模型擅长生成，但容易出现事实幻觉
- 如果没有可靠资料约束，输出内容可能看起来通顺但不准确

## 第 3 周：知识库与问答系统
- 传统问答系统依赖预先整理好的知识库
- RAG 的核心思路是：先检索，再生成
- 适合用在课程资料问答、教学资源整合、学习辅导场景

## 第 5 周：AI 在高校教学中的应用
- 可以用于答疑、资料整理、学习路径推荐
- 也可能带来引用不规范、过度依赖、事实错误等问题

## 我自己的想法
- 如果每门课都能有一个课程知识库，学生复习时会更方便
- 但前提是资料来源要可靠，不能让 AI 自己乱编
- 这个方向适合写“应用分析”类论文，不一定要真的做系统开发
