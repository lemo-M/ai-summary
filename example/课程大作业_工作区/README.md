# 示例工作区说明

这个文件夹是给 `notes/03_第一次任务实操_课程论文版.md` 配套的真实示例。

你可以直接拿它做三件事：

1. 对照教程看每个文件应该放什么内容
2. 在 VSCode 里打开整个文件夹，直接截图工作区树
3. 让 Claude Code 或 CCSwitch 基于这些文件继续往下生成新版本

当前结构：

```text
课程大作业_工作区/
  input/
    assignment.md
    notes.md
    references.md
  output/
    大纲_v1.md
    初稿_v1.md
  task.md
```

建议下一步：

1. 先让 AI 读取 `input/` 和 `task.md`
2. 再让 AI 继续改 `output/大纲_v1.md`
3. 最后按你的课程要求继续扩写或重写 `output/初稿_v1.md`
